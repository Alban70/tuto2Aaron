Présentation SASS

SASS est un pré-processeur qui intègre tout un tas de fonctionnalités qui vont permettrent d'écrire du CSS de manière plus organisée et plus propre.

Séparer en plusieurs feuilles de styles

SCSS

                    
                        @import url('https://fonts.googleapis.com/css2?family=Nunito&display=swap');
                        @import './partials/tuto'; // Uniquement pour la mise en page du tuto
                        
                        html {
                            font-family: 'Nunito', sans-serif;
                            background-color: rgb(250, 250, 250);
                        }
                        
                        body {
                            margin-bottom: 3rem;
                        }
                        
                        @import './chapters/ch1-p1';
                        @import './chapters/ch1-p2';
                        @import './chapters/ch1-p3';
                        @import './chapters/ch1-p4';
La directive @import permet de séparer vos feuilles de style de manière propre et de toutes les réunir en un seul fichier lors de la compilation. Ainsi, vous pouvez séparer les classes pour chaque style que vous développez: header ou footer par exemple. Il est également possible d'importer directement depuis une url() comme dans notre exemple depuis l'API Google Font.
