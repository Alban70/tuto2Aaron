Imbrication des propriétés: Nested Rules

Rendu visuel

th th
td td
td td

CSS

                    
                        table {
                            width: 100%;
                        }
                        
                        table th {
                            background-color: #ccc;
                            color: #fff;
                        }
                        
                        table td {
                            border: solid 1px #0044ff;
                            color: #ff0000;
                        }

SCSS

                    
                        table {
                            width: 100%;
                            
                            th {
                                color: #FFF;
                                background-color: #CCC;
                            }
                            
                            td {
                                border: solid 1px #0044ff;
                                color: #ff0000;
                            }
                        }
Pour éviter de répéter plusieurs fois la propriété table, il est possible en SCSS d'imbriquer les propriétés les unes à l'intérieur des autres. Il est conseillé de ne conserver au maximum que deux niveaux d'imbrication afin de conserver un code lisible.

Rendu visuel

CSS

                    
                        .button {
                            background-color: #5190ee;
                            border-radius: 5px;
                            border: none;
                            color: white;
                            cursor: pointer;
                            padding: .5rem;
                        }
                        
                        .button:hover {
                            background-color: #204d8f;
                        }

SCSS

                    
                        .button {
                            background-color: #5190ee;
                            border-radius: 5px;
                            border: none;
                            color: white;
                            cursor: pointer;
                            padding: .5rem;
                            &:hover {
                                background-color: #204d8f;
                            }
                        }
Même principe pour modifier le comportement d'une pseudo-classe grâce au sélecteur & qui doit se lire de cette manière: De classe .button ET :hover
