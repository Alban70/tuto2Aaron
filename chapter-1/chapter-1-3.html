Présentation SASS

SASS est un pré-processeur qui intègre tout un tas de fonctionnalités qui vont permettrent d'écrire du CSS de manière plus organisée et plus propre.

Héritages

Rendu visuel

CSS

                    
                        .button {
                            background-color: #5190ee;
                            border-radius: 5px;
                            border: none;
                            color: white;
                            cursor: pointer;
                            padding: .5rem;
                        }
                        
                        .button-cta {
                            background-color: #5190ee;
                            border-radius: 5px;
                            border: none;
                            color: white;
                            cursor: pointer;
                            padding: .5rem;
                        }

SCSS

                    
                        .button {
                            background-color: #5190ee;
                            border-radius: 5px;
                            border: none;
                            color: white;
                            cursor: pointer;
                            padding: .5rem;
                        }
                        
                        .button-cta {
                            @extend .button;
                        }
Notre bouton .button-cta possèdent les mêmes propriétés que .button. En SCSS, vous pouvez étendre les propriétés d'une classe à une autre grâce à la règle @extend.

L'exemple donné est particulièrement sans intérêt. L'avantage de @extend une classe est de la rendre la plus générique possible pour ensuite surcharger les propriétés qui nous intéressent.

Rendu visuel

SCSS

                        
                            .button__generic {
                                border-radius: 5px;
                                border: none;
                                cursor: pointer;
                                padding: .5rem;
                            }
                            
                            .button__extended {
                                @extend .button__generic;
                                background-color: #f88644;
                            }
                            
                            .button-cta__extended {
                                @extend .button__generic;
                                background-color: #73ec79;
                            }
Cela nous permet d'attribuer une couleur différente à chaque boutons sans avoir à redéfinir les propriétés border-radius, border, cursor, padding.

SCSS

                        
                            %button {
                                border-radius: 5px;
                                border: none;
                                cursor: pointer;
                                padding: .5rem;
                            }
                            
                            .button__extended {
                                @extend %button;
                                background-color: #f88644;
                            }
                            
                            .button-cta__extended {
                                @extend %button;
                                background-color: #73ec79;
                            }
En remplaçant la classe .button-generic par %button, vous signalez au compilateur Gulp de ne pas ajouter celle-ci dans votre CSS final. Seules les classes qui étendent de cette dernière se verront attribués les propriétés définies.
