Les Variables

Rendu visuel

CSS

                    
                        .button-var {
                            background-color: #5190ee;
                            padding: 20px;
                            border: none;
                            border-radius: 5px;
                            color: white;
                            cursor: pointer;
                        }

                        .button-var:hover {
                            background-color: #3e5e8f;
                        }
                        
                        @media only screen and (max-width: 768px) {
                            .button-var {
                                width: 100%;
                            }
                        }

SCSS

                    
                        $color-primary: #5190ee;
                        $padding: 10px;
                        $mobile-break: 768px;
                        $mobile-break: 900px !default; // Prend la valeur 768px sauf si une variable du même nom existe déjà
                        $mobile: "only screen and (max-width: #{ $mobile-break })";
                        
                        .button-var {
                            background-color: $color-primary;
                            padding: $padding + 10px;
                            border: none;
                            border-radius: 5px;
                            color: white;
                            cursor: pointer;
                        
                            &:hover {
                                background-color: darken($color-primary, 20);
                                // background-color: lighten($color-primary, 20);
                                // background-color: rgba($color-primary, .5);
                            }
                            
                            @media #{ $mobile } {
                                width: 100%;
                            }
                        }
Les variables en SCSS se déclarent avec le sélecteur $. Elles peuvent être de tous types, à vous de les utiliser dans le bon contexte. Si une variable doit être intégré dans une chaîne de caractères, il faudra l'interpoler de cette manière #{ $variable }.

Le drapeau !default permet d'assigner une valeur par défaut à une variable si elle n'existe pas déjà ailleurs. Peu importe l'ordre de priorité classique en CSS, c'est à dire que la dernière propriété prend habituellement le dessus.
