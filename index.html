Pré-requis

  • Node.js - Afin de pouvoir utiliser le gestionnaire de paquets NPM.
  • Gulp - Outil de type "toolkit" permet la compilation et l'automatisation de tâches.

Installation

  • Ouvrir un terminal et se positionner dans le dossier courant de notre projet "tuto-scss".
  • Exécuter la commande "npm install" pour installer tous les packages.
  • Exécuter "gulp" pour initialiser les fichiers css/js à vide.
  • Exécuter "gulp watch" pour surveiller et compiler les fichiers scss/js à chaque modification.
